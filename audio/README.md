# Background Music Setup

## Required Audio File

Please add a devotional music file to this directory:

**Filename:** `jai-shree-shyam.mp3`

### Suggested Music Options:
1. **Jai Shree Shyam Aarti** - Soothing devotional song
2. **Hanuman Chalisa** - Traditional prayer
3. **Khatu Shyam Bhajan** - Devotional hymn

### Where to Find:
- Download from YouTube (use a converter tool)
- Search for "Jai Shree Shyam Aarti MP3"
- Or "Khatu Shyam Ji Bhajan MP3"

### File Requirements:
- **Format:** MP3
- **Duration:** 3-5 minutes (will loop automatically)
- **Quality:** 128kbps or higher
- **Volume:** Pre-normalized (not too loud)

### How to Add:
1. Download or convert the audio file
2. Rename it to `jai-shree-shyam.mp3`
3. Place it in this `audio/` folder
4. The music will auto-play when users select the Spiritual theme

## Current Setup:
- Music player appears ONLY in Spiritual theme
- Has play/pause button
- Volume control slider
- Loops continuously
- Respects browser auto-play policies

## Note:
The website is already configured to play the music. You just need to add the MP3 file!
